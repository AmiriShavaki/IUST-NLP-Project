\Section
{آماده‌سازی محیط مجازی برای اجرای پروژه}
{
شما می‌توانید به کمک \lr{conda} و از طریق فایل \lr{requirements.txt} که در \lr{root} پروژه در اختیار شما قرار داده شده، به راحتی و فقط با اجرای دستور زیر تمامی پکیج‌های موردنیاز برای اجرای پروژه را نصب کنید:
\lr{
conda create --name <envname> --file requirements.txt
}
}
